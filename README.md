## About
Manage your dotfiles across multiple diverse machines, securely.

## Documentation
[chezmoi.io](https://chezmoi.io/)

## Install
```
sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply maxdzyubak
```
## Update
```
chezmoi update
```
